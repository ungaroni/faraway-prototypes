Faraway prototypes

Three concept directions for the ambient-sound app. Each one is a working interactive prototype — pick a flow, tap through it on desktop or mobile.

v1 · mood

Mood-based

Onboarding with goal pick (Sleep / Focus / Relax / Meditate), mood-organized library, end-of-session reflection.

 v2 · strategy applied

Tighter onboarding + retention loop

8-screen onboarding, "Tonight's sound" daily hook on home, Insights tab with weekly Soundprint, end-of-session reflection.
→
 v3 · teleport

Place-based atlas

"Where do you want to go?" — places not moods. Cosmic starfield home, atlas worlds (Aqua / Cozy / Wild / Weather / Night), live atmosphere-specific animations in the player, weekly Soundprint as a map of your journeys.
→
 v4 · journal

Journal — goal + stat + check-in

Animated cosmos splash → goal pick → one matched peer-reviewed stat → today's check-in (1–5 scale) → prompt → place player → journal-style paywall (paper, handwritten serif, stamp). Plus the cosmic home with "tonight's place" and recent journeys.
→
 v5 · research-backed

To feel · to leave

Built on real lifestyle-app research. Cosmos → value → prompt → magic → "why this place tonight?" (to feel / to leave) → save your first trip → when do you want this → value recap → combined account+paywall → notification prime → home. Simplified player (no sliders/insights yet).
→
 v6 · the beautiful one

Info card + Mood Meter + 3D Atlas

Psychology-grounded. Describe a place → poetic research card ("why this works") → "how does tonight feel?" (5-state Mood Meter grid) → build → player → save to Atlas → cosmic-Earth signup & paywall (rotating Earth in space) → home. New 3D-feeling Atlas tab with a slowly spinning Earth and gold pins for places.
